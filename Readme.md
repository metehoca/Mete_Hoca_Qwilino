# Mete Hoca Qwilino

A single unified library that simplifies the control and integration of **Qwilino** and **Arduino Modulino®** I2C (Qwiic/STEMMA QT) modules. 

Compatible with a wide range of microcontrollers including Arduino Uno/Nano R3, Uno/Nano R4, XIAO, ESP32, STM32, and other standard boards.

## Features
* **Unified API:** Control multiple sensors and modules under a single, clean library structure (`Qwilino.h`).
* **Cross-Platform:** Tested and fully compatible with AVR, ARM, and ESP architectures.
* **Plug & Play:** Designed for seamless use with the Qwiic/I2C ecosystem.

## Installation

### Via Arduino Library Manager (Recommended)
1. Open the Arduino IDE.
2. Go to **Sketch** -> **Include Library** -> **Manage Libraries...**
3. Search for **Mete Hoca Qwilino**.
4. Click **Install**. (When prompted, choose "Install all" to ensure all dependent sensor libraries are also downloaded).

### Manual Installation
1. Download the latest release `.zip` file from the [Releases](https://github.com/metehoca/Mete_Hoca_Qwilino/releases) page.
2. In the Arduino IDE, go to **Sketch** -> **Include Library** -> **Add .ZIP Library...**
3. Select the downloaded file.
